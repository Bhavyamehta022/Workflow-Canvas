#Workflow Canvas

---

## Features

- **Fetch workflows**: Creates a canvas where a workflow structure can be rendered.
- **Drag & Drop**: Side panel with available nodes that can be dragged onto the canvas.
- **Connect Nodes**: Create edges between nodes to define workflow logic.
- **Responsive Canvas**: Zoom, pan, and view workflow efficiently with React Flow controls.
- **Placeholder nodes**: Fallback nodes if the API returns an empty workflow.
